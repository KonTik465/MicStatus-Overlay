# MicStatus-Overlay
# MicStatus Overlay 🎙️  A lightweight Windows utility to track the **physical** state of your microphone.  ### Why? Many laptops have a dedicated button to mute the microphone. However, apps like Discord or Telegram often fail to detect this hardware-level mute, showing your mic as "active" when it's actually disabled.
